# Sanity Check 리스트

## 1. 시장가격 비교 (상장사)
- 괴리율 50% 초과: 데이터/가정 재확인 권고
- 괴리율 100% 초과: 입력 데이터 반드시 재검토

## 2. WACC 범위
- 정상: 6~15%. <5%: rf/beta 확인. >20%: D/E, ERP 확인.

## 3. 멀티플 범위
- EV/EBITDA 5~20x, P/E 8~30x, P/BV 0.5~5x

## 4. 교차검증
- SOTP vs DCF 괴리 >30%: 성장률 가정 점검

## 5. 시나리오
- 확률 합계 = 100%, DLOM: 비상장 10~30%/상장 0~5%
- 성장률 > WACC는 비현실적
